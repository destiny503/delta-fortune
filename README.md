**Колесо Фортуны**

Fullstack-приложение, разработано мной с нуля до Production под заказ в сжатые сроки.<br>

- Лёгкий фронтенд на чистом JavaScript с мини библиотеками.<br>
- Бэкенд на Node.js + Express.js с авторизацией и корректным распределением подарков.<br>
- База данных PostgreSQL, работа через Prisma ORM.<br>
- Успешное развёртывание на PaaS (Amvera).<br>
