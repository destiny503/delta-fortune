**Колесо Фортуны**

Fullstack-приложение, разработано мной с нуля до Production под заказ в сжатые сроки.

- Лёгкий фронтенд на чистом JavaScript с мини библиотеками.
- Бэкенд на Node.js + Express.js с авторизацией и корректным распределением подарков.
- База данных PostgreSQL, работа через Prisma ORM.
- Успешное развёртывание на PaaS (Amvera).
